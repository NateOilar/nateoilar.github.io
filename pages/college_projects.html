Liminal Forest

This was my first ever time creating anything in UE5 ever and it was really fun to experimnet and tinker around with procedurally generated content, material creation, and 3D modeling from scratch within UE5. With the grass and trees within the environment, are (PG) content that we learned to implement in class. There were issues at first with placing the assets and it generating to many of them and crashing the engine. making smaller areas to be affected helped with this issue and allowed the enironment to run in a more stable fashion. I created custom materials for this project as well and added them to the playground within the environment. With reguards to the playground, that is an asset I created entirly from scratch for this environment.
